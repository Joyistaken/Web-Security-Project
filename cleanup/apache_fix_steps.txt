COMPREHENSIVE STEPS TO FIX APACHE:

1. RESTORE ORIGINAL CONFIGURATION:
   - Locate backup file: C:\xampp\apache\conf\httpd.conf.original
   - Rename it back to httpd.conf: 
     Copy-Item "C:\xampp\apache\conf\httpd.conf.original" "C:\xampp\apache\conf\httpd.conf" -Force

2. MODIFY PHP.INI (FIX DUPLICATE MODULES):
   - Open C:\xampp\php\php.ini in a text editor
   - Find and comment out one of these lines by adding a semicolon at the beginning:
     extension=openssl
     extension=php_openssl.dll
   - Save the file

3. CHANGE APACHE PORT (AVOID CONFLICTS):
   - Open C:\xampp\apache\conf\httpd.conf
   - Find the line: Listen 80
   - Change it to: Listen 8080
   - Save the file

4. UPDATE VHOST CONFIGURATION:
   - Open C:\xampp\apache\conf\extra\httpd-vhosts.conf
   - Change all instances of <VirtualHost *:80> to <VirtualHost *:8080>
   - Save the file

5. UPDATE HOSTS FILE:
   - Run Notepad as administrator
   - Open C:\Windows\System32\drivers\etc\hosts
   - Add these lines:
     127.0.0.1 project.localhost.com
     127.0.0.1 websec.local.com
   - Save the file

6. FIX PERMISSIONS:
   - Run Command Prompt as Administrator
   - Run: icacls "C:\xampp" /reset /T
   - Run: icacls "C:\xampp\apache\logs" /grant "Everyone:(OI)(CI)F" /T

7. CLEAN APACHE LOGS:
   - Delete all files in C:\xampp\apache\logs
   - Or rename them to .old extension

8. DISABLE SKYPE OR OTHER SERVICES:
   - Close Skype if running (it uses port 80)
   - Stop IIS if installed: net stop w3svc

9. RESTART COMPUTER:
   - Restart your computer to clear any locked processes

10. RUN XAMPP AS ADMINISTRATOR:
    - Right-click XAMPP Control Panel and select "Run as administrator"
    - Start Apache

11. CHECK FIREWALL SETTINGS:
    - Make sure Windows Firewall allows Apache (httpd.exe)
    - Temporarily disable antivirus to test if it's interfering

12. IF ALL ELSE FAILS:
    - Reinstall XAMPP from scratch (after backing up your project files)
    - Or try different software like WampServer or Laragon

ACCESS YOUR SITES AT:
- http://project.localhost.com:8080
- http://websec.local.com:8080
