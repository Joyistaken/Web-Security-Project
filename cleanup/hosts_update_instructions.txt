HOSTS FILE UPDATE INSTRUCTIONS:

1. Run Notepad as administrator (right-click on Notepad in Start menu and select "Run as administrator")
2. Open the file: C:\Windows\System32\drivers\etc\hosts
3. Add these lines at the end of the file:

127.0.0.1 project.localhost.com
127.0.0.1 websec.local.com

4. Save the file and close Notepad
5. Restart Apache in XAMPP Control Panel

After these changes, you'll be able to access:
- http://project.localhost.com (for the project at C:/Users/esam3/Downloads/websec-main/web/public)
- http://websec.local.com (for the project at C:/xampp/apache/conf/ssl/websec/public)
